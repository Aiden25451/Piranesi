# Piranesi Monorepo

Fast serverless landing page (Hono + AWS Lambda) and a Next.js resume site.

## Stack

| Layer | Tech |
|---|---|
| **Monorepo** | pnpm workspaces |
| **Landing** | Hono, esbuild, Auth0, Lambda |
| **Resume** | Next.js, React, Tailwind CSS |
| **Infra** | Terraform, API Gateway, Lambda |
| **CI/CD** | GitHub Actions + OIDC |
